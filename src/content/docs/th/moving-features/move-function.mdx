## Intent

นำฟังก์ชันที่เมื่ออ่านแล้วพูดถึง *อ็อบเจกต์อื่น* เป็นส่วนใหญ่ — ฟิลด์ของมัน เมธอดของมัน กฎของมัน — แล้วย้ายฟังก์ชันนั้นไปยังอ็อบเจกต์ดังกล่าว บ้านใหม่เป็นเจ้าของข้อมูลที่ฟังก์ชันต้องใช้ ฟังก์ชันจึงเรียบง่ายขึ้น และการ coupling ระหว่างสองคลาสก็ลดลง

## The smell

นี่คือยารักษา **Feature Envy**: เมธอดที่ดูเหมือนสนใจคลาสอื่นมากกว่าคลาสที่มันอาศัยอยู่ สัญญาณบ่งชี้คือรายการพารามิเตอร์และตัวบอดี้ ถ้าฟังก์ชันรับอาร์กิวเมนต์ `plan` แล้วอ่าน `plan.tier`, `plan.dailyRate` และ `plan.discount` ในขณะที่แทบไม่แตะอ็อบเจกต์ของตัวเอง แสดงว่ามันอิจฉา `plan` ย้ายมันไปที่นั่น แล้วการเอื้อมข้ามไปเรียกก็จะกลายเป็นการเข้าถึงฟิลด์ธรรมดา

## Before → After

ค่าธรรมเนียมเบิกเกินบัญชี (overdraft charge) ที่อาศัยอยู่บน `Account` แต่คำนวณเกือบทั้งหมดจาก `plan` ของบัญชี ก่อนหน้านี้ ฟังก์ชันเอื้อมเข้าไปใน plan เพื่อเอาทุกอย่าง หลังจากนั้น มันอาศัยอยู่บน plan ซึ่งเป็นที่ที่ข้อมูลอยู่อยู่แล้ว

## Mechanics

1. ตรวจสอบทุกสิ่งที่ฟังก์ชันใช้ในบ้านปัจจุบันของมัน และยืนยันว่าส่วนใหญ่เป็นของคลาสปลายทาง ทำรายการสิ่งไม่กี่อย่างที่ยังมาจากต้นทาง
2. ตรวจสอบว่าองค์ประกอบฝั่งต้นทางเหล่านั้นควรเดินทางไปกับฟังก์ชัน หรือถูกส่งเข้ามาเป็นพารามิเตอร์
3. คัดลอกฟังก์ชันเข้าไปในคลาสปลายทาง และปรับบอดี้ให้ใช้ฟิลด์ของปลายทางเองโดยตรง
4. คอมไพล์และแก้ไขการอ้างอิงต่าง ๆ — ปลายทางอาจต้องมีพารามิเตอร์ใหม่สำหรับข้อมูลต้นทางที่ยังเหลืออยู่
5. เปลี่ยนฟังก์ชันเดิมให้เป็นตัวมอบหมาย (delegator) บาง ๆ ที่เรียกบ้านใหม่ หรือเปลี่ยนผู้เรียกมันโดยตรง
6. **รันเทสต์ของคุณ** พฤติกรรมต้องไม่เปลี่ยนแปลง
7. เมื่อผู้เรียกทุกตัวผ่านตำแหน่งใหม่แล้ว ให้ลบ delegator ออกหากมันไม่คุ้มกับการมีอยู่อีกต่อไป

## When to use / trade-offs

ย้ายฟังก์ชันเมื่อมันเอื้อมข้ามไปยังอ็อบเจกต์อื่นอย่างสม่ำเสมอ เมื่อหลายฟังก์ชันบนคลาสหนึ่งจะเรียบง่ายขึ้นหากจัดกลุ่มไว้บนอีกคลาส หรือเมื่อการย้ายมันทำให้คุณลบพารามิเตอร์ที่มีไว้เพื่อขนข้อมูลเข้ามาเท่านั้นได้

ต้นทุนคือผู้เรียกอาจจำเป็นต้องมี reference ไปยังอ็อบเจกต์ปลายทาง และฟังก์ชันที่ถูกย้ายอย่างกระตือรือร้นเกินไปอาจทำให้ตรรกะที่เกี่ยวข้องกระจัดกระจาย สิ่งตรงข้ามก็คือการย้ายมันกลับ: หากภายหลังการเปลี่ยนแปลงทำให้ฟังก์ชันกลับพึ่งพิงบริบท *เดิม* ของมันมากกว่า ก็ Move Function มันกลับบ้านอีกครั้ง
