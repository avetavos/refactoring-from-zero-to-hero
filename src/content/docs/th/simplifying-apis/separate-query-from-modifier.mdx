## เจตนา

เมื่อฟังก์ชันหนึ่งตอบคำถาม *และ* เปลี่ยนแปลงโลกในฐานะผลข้างเคียงไปด้วย ผู้เรียกใช้ก็สูญเสียอิสระที่จะแค่ถามเฉย ๆ ให้แยกงานสองอย่างนี้ออกจากกัน: **query** ที่คืนค่าและไม่แตะต้องอะไรเลย และ **command** ที่ดำเนินการเปลี่ยนแปลงและไม่คืนค่าอะไร ตอนนี้ผู้เรียกใช้ที่ต้องการแค่คำตอบก็สามารถเรียก query ได้โดยไม่ไปกระตุ้นผลข้างเคียง

## กลิ่นไม่ดี

คุณอยากเรียกฟังก์ชันเพื่ออ่านผลลัพธ์ของมันเท่านั้น — อาจในเทสต์ ในบรรทัด log หรือใน assertion — แต่คุณทำไม่ได้ เพราะการเรียกมันยังส่งอีเมล เปลี่ยนค่าฟิลด์ หรือเลื่อนเคอร์เซอร์ไปด้วย ลายเซ็นของมันโกหกเรื่องต้นทุน: มันดูเหมือนคำถามแต่แอบเป็นการกระทำ ฟังก์ชันใดก็ตามที่คืนค่าแต่ก็มีผลข้างเคียงที่สังเกตเห็นได้ คือผู้เข้าชิงสำหรับท่านี้

## ก่อน → หลัง

ฟังก์ชันสแกนคะแนน คืนค่าสูงสุด และ — ที่ฝังอยู่ภายใน — ยิงการแจ้งเตือนเมื่อคะแนนสูงสุดข้ามเกณฑ์ เราแยกมันออกเป็น query ที่บริสุทธิ์ชื่อ `highScore` และ command ชื่อ `alertOnHighScore`

## กลไก

1. คัดลอกฟังก์ชันแล้วตั้งชื่อสำเนาให้เป็น query ที่บริสุทธิ์ — ตามธรรมเนียมจะเป็นคำนามหรือชื่อสไตล์ `get`/`is` ที่สัญญาว่าให้แค่คำตอบ
2. ตัดผลข้างเคียงทุกอย่างออกจากสำเนา query โดยเหลือไว้แค่ค่าที่มันคำนวณ ให้มันคืนค่านั้นกลับมา
3. หาผู้เรียกใช้แต่ละจุดของฟังก์ชันเดิม จุดที่ผู้เรียกใช้ใช้ค่าที่คืนกลับมา ให้สลับไปใช้ query
4. ปล่อยให้ฟังก์ชันเดิมทำแค่ผลข้างเคียงของมัน ภายในมัน ให้เรียก query ตัวใหม่แทนที่จะคำนวณค่าใหม่
5. **รันเทสต์** หลังจากย้ายผู้เรียกใช้แต่ละจุดข้ามไปแล้ว
6. ถ้าฟังก์ชันเดิมตอนนี้คืนค่าที่ไม่มีใครใช้ ให้เปลี่ยนมันให้ไม่คืนค่าอะไร เพื่อให้ลายเซ็นของ command ซื่อสัตย์

## เมื่อไรควรใช้ / ข้อแลกเปลี่ยน

ใช้ท่านี้เมื่อใดก็ตามที่ฟังก์ชันที่มีค่าคืนกลับยังเปลี่ยนสถานะ ทำ I/O หรือยิงอีเวนต์ด้วย — และโดยเฉพาะก่อนคุณจะเขียนเทสต์ เพราะ query ที่บริสุทธิ์ตรวจ assert ได้ง่ายมาก การแยกที่สะอาดยังทำให้ query ปลอดภัยที่จะเรียกซ้ำ ๆ หรือ cache ได้

ต้นทุนคือผู้เรียกใช้ที่ต้องการทั้งสองพฤติกรรมจริง ๆ ตอนนี้ต้องเรียกสองครั้ง และค่าอาจถูกคำนวณสองรอบ การคำนวณซ้ำนั้นมักจะถูก ถ้ามันไม่ถูก ให้คำนวณครั้งเดียวแล้วส่งผลลัพธ์ต่อไป บางการดำเนินการเป็นอะตอมโดยธรรมชาติ — `pop` ที่คืนค่าและลบในขั้นเดียว — และการบังคับให้แยกกันจะเชื้อเชิญ race condition ในกรณีนั้น ให้คงมันไว้ด้วยกันโดยตั้งใจ
