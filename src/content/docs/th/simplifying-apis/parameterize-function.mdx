## เจตนา

มองหาฟังก์ชันสองตัวขึ้นไปที่เนื้อในเหมือนกันยกเว้นค่าที่ฝังไว้ตายตัวค่าเดียว แล้วยุบมันให้เหลือฟังก์ชันเดียวที่รับค่านั้นเป็นพารามิเตอร์ ตอนนี้ตรรกะที่ซ้ำซ้อนอยู่ในที่เดียวเป๊ะ ๆ และความแตกต่างระหว่างตัวแปรเดิมกลายเป็นอาร์กิวเมนต์ที่มองเห็นได้ที่จุดเรียกใช้

## กลิ่นไม่ดี

นี่คือรสชาติหนึ่งของ **Duplicated Code** คุณเห็น `discountFivePercent` และ `discountTenPercent` นั่งอยู่ข้าง ๆ กัน เหมือนกันทุกอย่างยกเว้นตัวเลข หรือ `tierBronze` และ `tierSilver` ที่ต่างกันแค่ค่า threshold ทุกครั้งที่ตรรกะที่ใช้ร่วมกันต้องได้รับการแก้ คุณต้องจำให้ได้ว่าต้องไปแก้ทุก clone — และวันหนึ่งคุณก็จะลืมไปสักตัว ค่าคงที่ที่ฝังไว้ตายตัวคือสิ่งเดียวที่แปรเปลี่ยน มันจึงอยากเป็นอาร์กิวเมนต์ ไม่ใช่สำเนาของทั้งฟังก์ชัน

## ก่อน → หลัง

ฟังก์ชันสองตัวที่คำนวณราคาหลังหักส่วนลด ต่างกันแค่อัตรา หลังจากนั้น ฟังก์ชันเดียวรับอัตราเป็นพารามิเตอร์

## กลไก

1. เลือกฟังก์ชันที่เกือบเหมือนกันตัวหนึ่งมาเป็นแม่แบบ
2. เพิ่มพารามิเตอร์สำหรับค่าที่แปรเปลี่ยน ตั้งชื่อให้สื่อความหมายอย่าง `rate` หรือ `threshold` ไม่ใช่ `value`
3. ภายในเนื้อใน ให้แทน literal ที่ฝังไว้ตายตัวด้วยพารามิเตอร์ใหม่ **รันเทสต์ของคุณ** — พฤติกรรมควรไม่เปลี่ยนสำหรับกรณีเดิมของฟังก์ชันตัวนั้น เมื่อคุณส่งค่าคงที่เดิมของมันเข้าไป
4. สำหรับ clone ที่เหลือแต่ละตัว ให้เปลี่ยนเส้นทางผู้เรียกใช้ของมันไปยังฟังก์ชันที่ถูก parameterize แล้ว โดยส่ง literal เฉพาะของ clone นั้นเข้าไปเป็นอาร์กิวเมนต์
5. **รันเทสต์ของคุณ** หลังเปลี่ยนเส้นทางแต่ละ clone เพื่อให้ความล้มเหลวชี้ไปยังการเรียกใช้จุดเดียว
6. เมื่อไม่มีอะไรเรียกใช้ clone เดิมแล้ว ให้ลบมันทิ้ง
7. พิจารณาว่าพารามิเตอร์ใหม่ควรถูกตรวจสอบความถูกต้องหรือไม่ — ตัวอย่างเช่น อัตราส่วนลดที่อยู่นอกช่วงศูนย์ถึงหนึ่งน่าจะเป็นบั๊กที่ควรปฏิเสธ

## เมื่อไรควรใช้ / ข้อแลกเปลี่ยน

หยิบท่านี้มาใช้เมื่อฟังก์ชันต่างกันแค่ค่าคงที่ เมื่อคุณกำลังจะ copy-paste ฟังก์ชันแล้วปรับตัวเลขตัวเดียว หรือเมื่อชุดของตัวแปรที่มีชื่อต่าง ๆ เพิ่มขึ้นเรื่อย ๆ การ parameterize ฆ่าความซ้ำซ้อนและทำให้มิติที่แปรเปลี่ยนชัดเจนและค้นพบได้

ข้อแลกเปลี่ยนคืออาร์กิวเมนต์แบบ literal ที่จุดเรียกใช้อาจอธิบายตัวเองได้น้อยกว่าฟังก์ชันที่ตั้งชื่อมาดี — `0.05` เปล่า ๆ บอกอะไรน้อยกว่า `priceWithFivePercentOff` ถ้าชุดของค่ามีจำนวนน้อยและสื่อความหมาย ค่าคงที่ที่มีชื่อหรือ enum ที่จุดเรียกใช้ก็คืนความชัดเจนนั้นกลับมา และจงต้านทานการ parameterize มากเกินไป: ฟังก์ชันที่มีปุ่มปรับแปดปุ่มใช้ยากกว่า clone ที่มันมาแทนที่
